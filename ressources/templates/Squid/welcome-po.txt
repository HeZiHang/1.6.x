<p>Artica proxy está disponível em duas versões: uma versão  gratuita e uma versão paga </p> 
<p> A  versão que descarregaste é uma versão simplificada.</p> 
<p>A aquisição da licença Artica  disponibiliza as seguintes funcionalidade:</p> 
<ul>
<li> A possibilidade de aumentar o tamanho da cache (limitado a 2 GB na versão gratuita) </li> 
<li>A possibilidade de criar regras de armazenamento em cache </li> 
<li> A possibilidade de ativar o modo multiprocessador </li> 
<li> A possibilidade de ativar o filtro DNS (ou seja possibilidade de filtrar os sítios web para solicitar o DNS). </li> 
<li> A possibilidade de guardar os logs </li> 
<li>Acesso a uma base de dados com 29 milhões de sítios web categorizados </li> 
</ul>
<p>É importante não esqueceres que um proxy é um intermediário.</p> 
<p> Vocês sabem que quando acedem  a uma página web por intermedio de um browser, este último conecta-se diretamente ao servidor para  aceder à página requisitada. Se vocês adicionarem um máquina (o proxy) entre o vosso computador e a internet, esta máquina vai registar as paginas web localmente e vai entregá-las ao seu computador sem que este aceda à internet.</p> 
<p>Em poucas palavras, isto vai permitir-lhes acelerar o acesso a web, controlar o acesso à web(por filtragem) e estar em conformidade com a legislação, conservando os logs durante o tempo legalmente exigido(que varia segundo o país).</p>
